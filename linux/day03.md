# 쉘(Shell)

1. 명령어 해석 기능(shell -> 커널)
2. 프로그래밍 기능
3. 사용자 환성 설정 기능

<h3>쉘 사용법</h3>

``

""

''

1. ;(세미콜론)

   여러개 명령어를 한줄에 실행함 왼쪽 -> 오른쪽

2. |(파이프)

   첫번째 명령어의 결과를 두번째 명령어에 추가해줌

3. &&

   첫번째 명령어가 정상적으로 실행됬을 때 두번째 명령어가 수행됨

4. ||

   첫번째 명령어가 오류날 경우 두번째 명령어 수행
