# TF-IDF (단어출현빈도 - 역 문서에서의 출현빈도)
- $tf(d,t)$ : 단어의 빈도수
- $idf(d,t)$ : 역 문서에서의 출현빈도
- $n_d$ : 전체 문서의 수
- $df(t)$ : 단어 t를 가진 문서의 수

$$ tf-idf(d,t) = tf(d,t) * idf(d,t) $$

$$ idf(d,t) = \log{\frac{n_d}{1 + df(t)}}$$

- 특수문자 제거
- 형태소 나누기
- 사전 원형으로 바꾸기
- 품사 기반으로 일부 품사의 형태소만 남기기
- 형태소 길이가 1을 초과하는 것만 남기기
- '배공빠르고 굿' -> ['배공', '빠르다', '굿']
