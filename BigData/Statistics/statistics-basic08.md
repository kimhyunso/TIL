# 확률
1. 덧셈
    P(A) + P(B) - P(A'교집합'B)
2. 곱셈
    P(B|A) = P(A'교집합'B)/P(A) => P(A) * P(B|A) = P(B) * P(A|B)

    **배반사건** : P(A '교집합' B) = 0
    
    **독립사건** : P(A '교집합' B) = P(A) * P(B)

# 표집분포
모수를 추정이 가능하다.

표집분포의 평균 == 모수의 평균 (같다)

표집분포의 분산 ==> 모분산 / 표본의 개수(n)

표집분포의 표준편차 ==> 편차 / 루트 N
