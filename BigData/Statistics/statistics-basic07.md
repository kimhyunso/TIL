# 평균 (m : 뮤)
시그마 : 표준편차

ex) 카드 뽑기 30, 60, 90

30 뽑을 확률 => 1/3

60 뽑을 확률 => 1/3

90 뽑을 확률 => 1/3

평균 => (30 * 1/3) + (60 * 1/3) + (90 * 1/3) = 60

# 표본의 평균 ˉx(x-bar)
ex) 카드 뽑기 30, 60, 90 / n = 2 (몇가지를 뽑을 것인가)

30 60 => 평균 : 45

30 90 => 평균 : 60

60 90 => 평균 : 75

90 30 => 평균 : 60

90 90 => 평균 : 90
