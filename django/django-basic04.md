# 가상 환경 설정
`python -m venv <appname>`

`source venv/Scripts/activate`

## 가상 환경 설정 연결 끊기
`deactivate`

# requirements.txt
가상 환경에서 설치한 목록을 저장한다.
`pip freeze < requirements.txt`
