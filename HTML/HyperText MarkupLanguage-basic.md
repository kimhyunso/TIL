# block과 inline
`block` : 다음 요소(element)가 다음줄로 내려감

`inline` : 다음 요소가 같은 줄에 진행됨

# web 1.0 web 2.0 web 3.0 차이
> web 1.0 : read only 참여불가
>
> web 2.0 : 사용자가 작성가능 (데이터 중앙화) 참여가능
>
> web 3.0 : 탈중앙화, 블록체인 (데이터 탈중앙화) 지분가능
