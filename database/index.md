# index
특정 데이터를 찾을려고 할때, 데이터양이 많다면 **FULL SCAN**을 하기 때문에 속도가 굉장히 느리다.

- FULL SCAN : 데이터를 위에서부터 마지막까지 순차적으로 찾는 것

# 예시

|이름|나이|
|-|-|
|홍길동|20|
|광개토대왕|25|
|이순신|35|
|손오공|20|

나이가 23인 데이터를 찾기위해 위에서 부터 순차적으로 데이터를 찾는다. - **FULL SCAN**

`SELECT * FROM tableName WHERE 나이 = 23;`
